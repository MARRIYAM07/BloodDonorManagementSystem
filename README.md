## Overview

This platform manages blood donation records, donor registrations, and blood donation campaigns.  
It allows hospitals, blood banks, doctors, and patients to efficiently track blood availability and connect with donors, making the process **faster, safer, and more transparent**.

---

## Features

- Donor registration after medical screening  
- Role-based access for hospitals and blood bank staff  
- Built-in timer restricting donors from donating too frequently  
- Search and request functionality for doctors, patients, and attendants  
- Campaign management and donor participation tracking  

---

## Modules

### 1. Donor & Blood Record Management
- Manage donor registration and blood records  
- Role-based access for hospitals and blood banks  
- Safety timer for donor intervals  
- Search for blood groups and request blood from donors  

### 2. Blood Donation Campaign Management
- Maintain records of blood donation campaigns  
- Track donor registrations and participation

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python/Flask
- **Database:** MS SQL
- **Other Tools/Libraries:** Python libraries for fetching data and connecting database with the interpreter

## Installation

1. Clone the repository:
   git clone https://github.com/MARRIYAM07/BloodDonorManagementSystem.git
2. Navigate to the project folder:
   cd BloodDonorManagementSystem
3. Install required Python libraries:
   pip install -r requirements.txt
4. Run the app:
   python app.py
